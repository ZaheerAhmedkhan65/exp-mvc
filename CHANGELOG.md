# Changelog

All notable changes to exp-mvc will be documented in this file.

## [1.2.0] - 2024-12-16

### Added
- Initial release of exp-mvc
- Project generation with MVC architecture
- Component generators (model, controller, route, service)
- Automatic dependency checking and installation
- File watcher for auto-dependency management
- EJS template support with Bootstrap
- Mongoose integration
- Joi validation support

### Features
- `expmvc new` - Create new projects
- `expmvc generate` - Generate components
- `expmvc watch` - Auto-dependency installation
- Built-in error handling middleware
- Ready-to-use project structure
- EJS template support with Bootstrap
- Mongoose integration
- Joi validation support
